# ScreenGuard

Otomatik ekran kilitleme uygulamasi. Bilgisayar basindan kalktiginizda ekraninizi otomatik olarak kilitler.

## Ozellikler

- **Yuz Tanima**: Kayitli yuzunuzu tanimadiginda ekrani kilitler
- **Hareketsizlik Algilama**: Fare/klavye hareketi olmadiginda ekrani kilitler
- **Bildirimler**: Kilitleme oncesi uyari bildirimi gonderir
- **Sistem Tepsisi**: Arka planda calisir, ayarlar tepsiden yonetilir

## Kurulum

### macOS
1. `ScreenGuard-macOS.dmg` dosyasini indirin
2. DMG dosyasini acin
3. ScreenGuard uygulamasini Uygulamalar klasorune surukleyin
4. Uygulamayi calistirin

### Windows
1. `ScreenGuard.exe` dosyasini indirin
2. Dogrudan calistirin veya istediginiz bir klasore tasiyip calistirin

## Kullanim

1. Uygulama baslatildiginda sistem tepsisinde gorulur
2. Tepsideki ikona tiklayarak ayarlara erisin
3. Yuz kaydetmek icin "Yuz Kaydet" secenegini kullanin
4. Yuz Tanima ve Hareketsizlik algilamayi acip kapatabilirsiniz

## Ayarlar

- **Yuz Tanima**: Acik/Kapali
- **Hareketsizlik**: Acik/Kapali  
- **Yuz Timeout**: Yuz gorunmeyince kac saniye sonra kilitlensin (10-60s)
- **Hareketsizlik Suresi**: Aktivite olmazsa kac saniye sonra kilitlensin (30-300s)

## Gereksinimler

- macOS 10.15+ veya Windows 10+
- Kamera (yuz tanima icin)
